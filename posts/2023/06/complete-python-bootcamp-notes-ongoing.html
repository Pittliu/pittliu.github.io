三大主流编程范式： 1. 面向过程(POP：(Procedure Oriented Programming):这种编程风格将程序结构分解成一系列步骤和算法,强调过程和函数的逻辑顺序。主要语言有C,Fortran等。处理简单的步骤与过程,可用于底层系统/广泛使用的语言 2. 函数式编程(FP：Functional Programming):这种编程风格通过组合简单的函数来实现复杂的计算,强调函数和函数之间的组合关系。主要语言有Haskell,Lisp等。强调函数复用与组合,适用于并发与分布式领域。 3. 面向对象(OOP：Object Oriented Programming):这种编程风格将程序结构分解成一个个封装属性和方法的对象,强调通过对象之间的消息传递和方法的调用来实现功能。主要语言有Python,Java,C++等。解决复杂问题,易扩展与维护,是现代主流的软件工程范式。

其他补充： 标记语言(markup language):像排版一样,使用标签来标记内容和格式。(如HTML, XML) 命令式(imperative):按步骤一条一条命令式地实现功能。(大部分语言都有这种范式) 逻辑语言(logical):像推理一样,用命题和逻辑来表达计算。(如Prolog) - JavaScript:动态脚本语言,多范式(imperative,functional,object-oriented)
- Ruby:面向对象的动态脚本语言,也支持函数式范式 
- C#:静态类型面向对象语言
- CSS:属于标记语言范式,用于HTML美化与布局
- BAT:指代互联网三大技术公司,并非一门编程语言

# List Comprehensions
Python有一个高级特性 called 列表推导式。它们允许快速构建列表。要完全理解列表推导式,我们需要理解for循环。所以如果您不完全理解这个部分,请不要担心,您可以跳过它,因为我们将在以后返回到这个主题。简单说,列表推导式是一种用简洁的方式创建列表的语法。它结合了for循环和if条件判断的功能。举个例子:没有列表推导式,我们可以这样创建一个列表:

result = []   # 创建一个空列表

for i in range(10): # 循环10次,i的值从0到9
    if i % 2 == 0:   # 判断i是否能被2整除,如果是偶数则条件满足
        result.append(i) # 将i添加到result列表

使用列表推导式,可以这样写:

python 
result = [i for i in range(10) if i % 2 == 0]

这产生了同样的结果,但代码更简洁清晰。列表推导式的一般形式是:  
[expr for iter_var in iterable if cond_expr]它包含:- 一个表达式expr,它会被计算后的结果会添加到新的列表中。  
- 一个迭代器iter_var,它会依次取iterable中的每个元素。  
- 一个条件表达式cond_expr,它用于过滤迭代元素。

first_col = [row[0] for row in matrix]

  1. Set and Booleans 在Python中:Set 是无序且不重复的集合。可以使用 { } 或者 set() 函数创建集合。
  2. 集合支持 operations 如 union、intersection、difference 等。
  3. 可以使用add、remove、discard、pop等方法修改集合。
  4. 可以使用in和not in来判断元素是否在集合中。例如:
python
s = {1, 2, 3}
s.add(4)
s.remove(2)
s.discard(5)  # 无操作

s.union({1, 2})   # {1, 2, 3, 4}
s.intersection({1, 2})  # {1, 2}
s.difference({1, 2})   # {3, 4}

2 in s   # False
2 not in s # True

Booleans 是True和False两种值。

python
True and True  # True
True and False # False 
not True   # False
True or False # True

布尔值常用于条件判断和逻辑表达式。例如: 

python 
if age > 0:   # 条件判断

age > 0 and age < 18   # 逻辑表达式

重要的文件方法有: - pwd - read() 读取整个文件
- readline() 读取一行
- readlines() 读取所有行到一个list
- write() 写入字符串
- close() 关闭文件
- seek() 移动文件读取指针 例如:

python
f = open('file.txt', 'r')

f.close()

lines = f.readlines()  # 读取所有行到list

使用with语句可以自动关闭文件:

python
with open('file.txt', 'r') as f:
    content = f.read()

文件自动关闭,无需调用f.close()

01-Python Comparison Operators

Comparison Operators

OperatorDescriptionExample
== If the values of two operands are equal, then the condition becomes true. (a == b) is not true.
!= If values of two operands are not equal, then condition becomes true. (a != b) is true
> If the value of left operand is greater than the value of right operand, then condition becomes true. (a > b) is not true.
< If the value of left operand is less than the value of right operand, then condition becomes true. (a < b) is true.
>= If the value of left operand is greater than or equal to the value of right operand, then condition becomes true. (a >= b) is not true.
<= If the value of left operand is less than or equal to the value of right operand, then condition becomes true. (a <= b) is true.
